## The Unified System

All three sites follow the same pattern:
1. `llms.txt` - Link index for discovery
2. `llms-full.txt` - Full content for LLM consumption
3. Direct `.md` download - Append `.md` to any doc URL
4. `docs_map.md` - Navigation structure (only discovered on Claude Code so far)
